# Time Tracker App

#### est. 4 July, 2018

## Project Setup

- `bundle install`
- `rails db:create && rails db:migrate`
- `rails db:seed`
- `rails s`

## Potential issues

- Routes not customized for not-signed-in users
- Slow rendering if database gets bigger
- Vulnerabilities not assessed
